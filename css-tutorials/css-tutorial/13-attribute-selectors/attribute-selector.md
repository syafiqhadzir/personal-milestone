# Notes:-

- ^=(value) : starts with or contain (value)
- $=(value) : ends with (value)
- \*=(value) : contains any (value)
- ~=(value) : contains specific (value)
- |=(value) : specified value or immediately followed by “-“

```css
h2[class|= g

selector {
 property: value;
 property2: value2;
}
```
