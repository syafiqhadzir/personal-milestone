**Terms**

1.  Functions

```javascript
function greet() {
    console.log('Hello World')
}
```

2.  Variables

```javascript
let greet = "Hello World"
```

3.  Statements

```javascript
greet()
```
4.  Parameters & Arguments

```javascript
function greet(name) { // parameters
    console.log('Hello ' + name)
}

greet('Syafiq') // arguments
```
