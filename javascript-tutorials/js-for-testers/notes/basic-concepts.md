**Basic Concepts**

1. **Variables**

   - String
   - Number
   - Boolean

2. **Constant**

Constants are simply variables whose value cannot be reassigned.

3. **Conditional Operators**

   - If
   - If Else
   - Else

4. **Loops**

   - While
   - For

5. **Functions**

Functions are a piece of code that takes some input, process it and return the output.
