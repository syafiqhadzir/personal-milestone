**Network Interface Naming Convetion**

Conventionally, network interfaces in Linux are enumerated sequentially as <kbd>eth0</kbd>, <kbd>eth1</kbd>, <kbd>eth2</kbd>, and so on.

|  NetIF  | Description |
|---------|-------------|
|  eth0   |   Ethernet  |
|  wlan0  |   Wireless  |
|  ppp0   |   Dial-Up   |
|  bond0  | First-Bonded |
|  virbr0 | Int. Bridge |
|   lo    |   Loopback  |
